PATHFINDER

Share your Waypoint in real time with your friends, without any server support.



ABOUT

Pathfinder makes it easier than ever to set waypoints in the world that your friends can see in real time.
Thanks to a unique distribution system, it is not necessary for this mod to be installed on the server.



FEATURES

  • Press: [SHIFT + Middle Mouse Button] to create a Waypoint.
  • Press: [P] to open the Menu.


REQUIREMENTS

Pathfinder is currently a Fabric only Mod so you have to install the Fabric API before adding Pathfinder to your game.

This Mod is: CLIENT SIDE

You however can install Pathfinder on your Server to use the normal netwokring system.



MISC

Feel free to include Pathfinder in your Modpack.

Since this mod is still very new I would be happy about all feedback.
